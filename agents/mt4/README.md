# Hedge Edge MT4 Agent

## Status: 🔜 Planned

MetaTrader 4 support is planned for a future release.

The MT4 agent will follow the same architecture as MT5:
- MQL4 Expert Advisor
- Native DLL for ZeroMQ + libsodium
- Same IPC protocol (ZMQ ports 51810/51811)

## Differences from MT5

- MQL4 vs MQL5 syntax differences
- 32-bit DLL required (MT4 is 32-bit only)
- Slightly different order/position handling
